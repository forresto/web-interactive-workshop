CSS

CSS rules can refer to 3 things: tag name, id, or class.

Tag names

Like body, h1, h2, h3, p, a, img

<style> a { /* all links will be red */ color: #FF0000; } a:hover { /* all links will be blue when the mouse is over */ color: #0000FF; } </style>

#id

Only one element can have a particular id.

<style> #maintitle { font-size: 25px; } </style> <h1 id="maintitle">This is the main title of the page.</h1>

.class

Many elements can have the same sytle. Each element can have more than one style.

<style> .large { font-size: 25px; } .blue { color: 25px; } </style> <p class="large">This is large.</h1> <p class="blue">This is blue.</h1> <p class="large blue">This is large and blue.</h1>
